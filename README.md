# Customer Behavior & Segmentation Analysis (Power BI Dashboard)

This project analyzes customer shopping behavior using transactional data from 3,900  purchases across various product categories. The goal is to uncover insights into spending  patterns, customer segments, product preferences, and subscription behavior to guide strategic  business decisions. 

# 📊 Data Analytics Project – End-to-End Workflow

## 🔍 Overview

This project demonstrates a complete data analytics workflow, starting from raw data ingestion to delivering actionable insights through dashboards and presentations. It covers data cleaning, exploratory data analysis (EDA), SQL querying, and business intelligence reporting.

The goal is to transform raw data into meaningful insights that support data-driven decision-making.

---

## 📁 Dataset

* The dataset used in this project contains structured data suitable for analysis.
* It includes multiple features such as categorical and numerical variables.
* Data is loaded and processed using Python for analysis and visualization.

## ⚙️ Project Steps

### 1. Data Loading

* Imported dataset into Python using Pandas.
* Inspected structure, data types, and basic statistics.

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and relationships between variables.
* Identified trends, patterns, and potential anomalies.
* Visualized data using charts and graphs.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicates and inconsistencies.
* Converted data types where necessary.
* Prepared dataset for analysis and reporting.

### 4. SQL Analysis (MySQL)

* Imported cleaned data into MySQL Server.
* Wrote SQL queries to:

  * Aggregate data
  * Filter and segment information
  * Extract key business insights

### 5. Dashboard Creation (Power BI)

* Built an interactive dashboard to visualize key metrics.
* Included charts, KPIs, and filters for better usability.
* Focused on clarity and business relevance.

### 6. Reporting

* Compiled insights into a structured report.
* Highlighted key findings and recommendations.

### 7. Presentation (Gamma)

* Created a professional presentation summarizing:

  * Problem statement
  * Approach
  * Key insights
  * Business impact

## ▶️ How to Run the Project

1. **Clone the Repository**

   ```bash
   git clone <your-repo-link>
   cd <project-folder>
   ```

2. **Install Dependencies**

   ```bash
   pip install -r requirements.txt
   ```

3. **Run Python Scripts / Notebook**

   * Open Jupyter Notebook and execute all cells for EDA and preprocessing.

4. **Set Up MySQL**

   * Import the cleaned dataset into MySQL Server.
   * Run the provided SQL queries.

5. **Open Power BI Dashboard**

   * Load the `.pbix` file in Power BI Desktop.

6. **View Presentation**

   * Open the Gamma presentation link or exported PPT.
